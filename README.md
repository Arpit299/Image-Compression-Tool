## Image Compression Tool
This project provides a simple and efficient way to compress and resize images directly in a Google Colab notebook using the Pillow library.

## Features
Quality Adjustment: Use a slider to control the JPEG compression quality (1-95).
Resizing: Scale down image dimensions by a ratio to further reduce file size.
Visual Comparison: Displays the original and compressed images side-by-side.
Batch Statistics: Shows original size, compressed size, and the percentage of reduction.
Easy Download: Automatically prompts to download the processed image.
## How to Use
Setup: Run the first cell to import necessary libraries and define the compression function.
Upload: Use the upload cell to select an image from your local computer.
Configure: Adjust the quality_setting and resize_setting sliders.
Process: Run the compression cell to see the results and compare visual quality.
Download: The final cell will trigger a browser download for your new compressed image.
## Requirements
Python 3.x
Pillow
Matplotlib
Google Colab environment
